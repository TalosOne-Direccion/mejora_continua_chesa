# Carpeta de Orígenes Crudos (/raw)

Coloca aquí todos los documentos de origen que quieras que yo (el Agente) procese e integre en la Wiki. 

Ejemplos de archivos que puedes guardar aquí:
- Transcripciones de reuniones.
- Requerimientos iniciales en archivos de texto.
- Artículos interesantes convertidos a Markdown.
- Notas rápidas sobre reglas de negocio.

**Nota**: Yo solo leo estos archivos, nunca los modifico. Tu información original siempre estará a salvo.
